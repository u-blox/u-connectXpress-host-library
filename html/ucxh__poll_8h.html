bool ucxhPARSER_setOnDataReceived ( int(*)(int channel, const uint8_t *buffer, size_t size)  callback)

Register callback to invoke upon reception of data. The callback is invoked independent of if the data is received in DATA mode or through the an EDM Data channel

Parameters
[in]callbackCallback to invoked upon reception of data.
[in]channelThe EDM Data channel, or -1 if data was received in DATA mode.
Returns
false if registration failed, otherwize true.
Return values
callbackThe callback shall return 0 if sucessful, < 0 if failed during processing.
